			JavaDate

3.14 (Date Class) Create a class called Date that includes three instance 
variables-a month (type int), a day (type int) and a year (type int). Provide 
a constructor that initializes the three instance variables and assumes that 
the values provided are correct. Provide a set and a get method for each instance
 variable. Provide a method displayDate that displays the month, day and year 
separated by forward slashes (/). Add a main method to the class that demonstrates 
class Date's capabilities.
