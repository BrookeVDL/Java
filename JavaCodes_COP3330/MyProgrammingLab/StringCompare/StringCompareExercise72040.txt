MyProgrammingLab; Java
COP 3330
Exercise 72040
			StringCompare.java
Write a program that uses String method regionMatches to compare 
two strings input by the user. The program should prompt the user 
to enter two strings, the starting index in the first string, the 
starting index in the second string, and the number of characters 
to be compared. The program should print whether or not the strings 
are equal. (Ignore the case of the characters during comparison.)
